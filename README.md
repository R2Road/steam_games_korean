스팀의 한국 게임 목록
</h1>



## 개요
* 스팀의 한국 게임 목록 입니다.



## 제보
* 상단에 링크된 Discord 에 남겨 주세요.
* 해당 게임과 직접적인 관련이 있으시면 아래 "필요 정보" 의 항목을 모두 알려 주세요.
* 해당 게임과 직접적인 관련이 없으시면 "스팀의 게임 페이지 링크" 를 알려주세요.



## 대상
* 플레이 가능한 버전이 "대중" 에게 공개된 적이 있는 한국산 게임.
* 한국산 게임의 정의
  * 한국인 또는 법인 소유의 개발사가 만든 게임
  * 최대 지분의 소유주가 한국인 또는 한국 법인 이면 한국 개발사로 인정( 예 : Nexon )
  * 해외 자회사 또는 해외 스튜디오가 개발해도 한국산 게임으로 인정.( 예 : 칼리스토 프로토콜 )
  * 게임이 외국에 팔려나가도 최초 출시가 한국 회사면 한국 게임으로 인정



## 필요 정보
* 스팀의 게임 페이지 링크
* 장르( 제작자가 명명한 장르 우선 사용, 없다면 스팀 태그 사용 )
* 사업자 등록 정보
  * 등록 번호( 국내 기업 확인용 )
  * 등록증에 기재된 개발사 이름( 국내 기업 확인용 )



## 작성 방법
* 저장소 내의 "game_list.ods"( libreoffice spreadsheet ) 파일을 연다.
* "develpoer" sheet에 개발사 정보를 기입 한다.
* "publisher" sheet에 퍼블리셔 정보를 기입 한다.
* "list" sheet에 게임 정보를 기입 한다.
* Macro 를 작동시켜 "README.md" 파일을 갱신한다.



## 정보 표시 규칙
* "게임링크" 는 제목에 링크를 걸고 주소를 직접 보이지 않는다.
* "개발사", "퍼블리셔" 가 같다면 하나로 표시한다.
